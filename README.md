# RemajaBijak

RemajaBijak adalah sebuah website edukatif yang dibuat untuk membantu mendapatkan informasi dan insight yang bermanfaat seputar edukasi seks, mitos vs fakta, quiz interaktif, serta artikel-artikel edukasi. Proyek ini ditujukan untuk masyarakat umum (terutama remaja) yang ingin belajar dengan lebih baik.
